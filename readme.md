## InvestCalc - Simulador de Investimentos B3

O **InvestCalc** é uma aplicação web moderna que permite simular a compra de ações da bolsa de valores brasileira (B3) em tempo real. O objetivo do projeto é demonstrar o uso de APIs externas, manipulação assíncrona de dados e criação de interfaces responsivas.

 Tecnologias Utilizadas

* **HTML5 & CSS3:** Estrutura e estilização moderna com foco em UX (User Experience) e Dark Mode.
* **JavaScript (ES6+):** Lógica de programação, cálculos e integração com API.
* **Brapi API:** Consumo de dados financeiros reais diretamente da bolsa de valores.

##  Funcionalidades

 [x] Busca de cotações em tempo real por Ticker (ex: PETR4, VALE3).
 [x] Cálculo automático de quantidade de cotas com base no valor disponível.
 [x] Cálculo de saldo remanescente após a "compra" simulada.
 [x] Interface totalmente responsiva (Desktop e Mobile).
 [x] Tratamento de erros para ativos não encontrados ou falhas de conexão.

## Como rodar o projeto

1. Clone este repositório.
2. Obtenha um token gratuito em [brapi.dev](https://brapi.dev/).
3. Substitua a variável `API_TOKEN` no arquivo `script.js`.
4. Abra o arquivo `index.html` no seu navegador.
